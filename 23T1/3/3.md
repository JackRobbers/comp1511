# variable name rules
##### Valid C: 
* can only contain letters, numbers, or underscores.
* must not start with a number.

##### Good style:
* use snake_case
* be descriptive or conventional  (e.g `i` in loops)
* #defines in SHOUTING_SNAKE_CASE.
